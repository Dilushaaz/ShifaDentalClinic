# Shifa Dental Clinic Management System

## Project Overview

Shifa Dental Clinic Management System is a web-based application developed using Django, HTML, CSS and SQLite. The system helps manage patient records, appointments and clinic expenses through a simple user interface.

## Features

* User Login Authentication
* Patient Registration
* Appointment Management
* Patient Records Management
* Expense Tracking
* Dashboard Overview

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite
